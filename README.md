# Portfolio Website

A modern, advanced portfolio website built with Next.js, TypeScript, and Tailwind CSS featuring animations, interactive charts, and a beautiful responsive design.

- **Modern Tech Stack**: Built with Next.js 14, TypeScript, and Tailwind CSS
- **Animated Hero Section**: Student with laptop illustration and floating particles
- **Interactive Tech Stack**: Icon-based technology showcase with hover effects
- **Advanced Skills Section**: 
  - Three viewing modes: Radar Chart, Categories, and Bar Graph
  - Visual skill proficiency representation
  - Categorized skill breakdown
- **Project Timeline**: Beautiful timeline layout showcasing projects with details
- **Experience Section**: Work experience, internships, and certifications
- **Contact Form**: Functional contact form with validation
- **Smooth Animations**: Framer Motion powered animations throughout
- **Fully Responsive**: Mobile-first design approach
- **Dark Mode**: Professional dark theme

## рџЊђ Deployment

Deploy to GitHub Pages:

1. Push your code to the `main` branch.
2. In the repository settings, set GitHub Pages source to `GitHub Actions`.
3. Every push to `main` will run the workflow and publish the latest static export.

The app is configured for static export, so it can be hosted on GitHub Pages without a separate server.
