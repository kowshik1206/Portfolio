# Portfolio Website

A modern, advanced portfolio website built with Next.js, TypeScript, and Tailwind CSS featuring animations, interactive charts, and a beautiful responsive design.

## 🚀 Features

- **Modern Tech Stack**: Built with Next.js 14, TypeScript, and Tailwind CSS
- **Animated Hero Section**: Student with laptop illustration and floating particles
- **Interactive Tech Stack**: Icon-based technology showcase with hover effects
- **Advanced Skills Section**: 
  - Three viewing modes: Radar Chart, Categories, and Bar Graph
  - Visual skill proficiency representation
  - Categorized skill breakdown
- **Project Timeline**: Beautiful timeline layout showcasing projects with details
- **Experience Section**: Work experience, internships, and certifications
- **Contact Form**: Functional contact form with validation
- **Smooth Animations**: Framer Motion powered animations throughout
- **Fully Responsive**: Mobile-first design approach
- **Dark Mode**: Professional dark theme

## 📦 Installation

1. Install dependencies:
\`\`\`bash
npm install
\`\`\`

2. Run the development server:
\`\`\`bash
npm run dev
\`\`\`

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## 🛠️ Technologies Used

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Charts**: Recharts
- **Icons**: React Icons
- **Utilities**: React Intersection Observer

## 🎨 Customization

- Update personal information in component files
- Modify colors in \`tailwind.config.ts\`
- Add/remove projects in \`Projects.tsx\`
- Update skills data in \`Skills.tsx\`
- Customize animations in individual components

## 📝 Sections

1. **Hero**: Introduction with animated illustration
2. **Tech Stack**: Technology icons with statistics
3. **Skills**: Interactive charts and categorized skills
4. **Projects**: Timeline-based project showcase
